# Notes App — MERN Stack

A Personal Notes app with Clerk authentication.

## Team
- Rohan — Backend foundation
- Pranav — API routes & auth middleware
- Niranjan — Frontend core & routing
- Sreya — Notes UI & styling

## Setup

### Backend
```
cd backend
npm install
npx prisma generate
npm run dev
```

### Frontend
```
cd frontend
npm install
npm run dev
```

## Environment Variables
make eg.env to .env in both backend/ and frontend/ and fill in your keys.
