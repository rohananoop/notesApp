# Notes App — MERN Stack

A Notes app with Clerk authentication.

## Team

- Sreya — Notes UI & styling

## Setup

### Backend
```
cd backend
```
npm install
npx prisma generate
npm run dev

### Frontend
cd frontend
npm install
npm run dev

## Environment Variables
Copy .env.example to .env in both backend/ and frontend/ and fill in your keys.
